# Data Provenance and Redistribution

The experiment used a file named `Churn_Modelling.csv`.

The original acquisition record and license were not retained. Although
similarly named copies may be publicly accessible, filename similarity
does not establish authoritative provenance or redistribution rights.

Accordingly:

- the original dataset is not included in this repository;
- no license is asserted for that file;
- public redistribution is deferred; and
- reproducibility should use a synthetic dataset or a source whose
  license and provenance have been verified.

A future synthetic sample should preserve only the schema and validation
conditions needed to exercise the framework. It should not reproduce
third-party records.
